# Sound Files

Place your MP3 files in this directory with the following names:

- **lava.mp3** - Background music for the intro page (looping)

## Recommended Sources for Free Sound Effects:

- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [BBC Sound Effects](https://sound-effects.bbcrewind.co.uk/)

## Tips:

- Keep files small (under 500KB) for quick loading
- Use short sounds (1-3 seconds)
- Adjust volume in app.js if sounds are too loud/quiet (currently set to 0.5)
